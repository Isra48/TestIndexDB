# TestIndexDB

## Formato de carga para participantes

El CSV de participantes usa una sola columna con el nombre del participante. Puedes incluir un encabezado opcional; se recomienda usar `Participante` o `Name` para que sea claro. Cada fila representa a una persona distinta.

Ejemplo de contenido:

```
Participante
Ana García
Luis Hernández
María Rodríguez
Carlos Pérez
Sofía López
```

Puedes reutilizar el archivo de ejemplo incluido en `templates/participants-template.csv` para comenzar a cargar los concursantes.
